# Projeto 1 - Programação Orientada a Objetos

## 📌 Descrição
Este projeto tem como objetivo a refatoração de um sistema estruturado para o paradigma de Programação Orientada a Objetos (POO), aplicando conceitos como encapsulamento e abstração.

## 🧠 Conceitos aplicados
- Encapsulamento (atributos privados)
- Classes e objetos
- Métodos de instância
- Separação de responsabilidades

## 📂 Estrutura do projeto
- src/models/cliente.py → classe Cliente
- src/main.py → testes do sistema

## ▶️ Execução
```bash
python src/main.py
